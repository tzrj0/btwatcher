## 简介

BT Watcher 是一个专为宝塔面板设计的实时日志监控工具，它可以帮助您：

- 🔍 实时监控网站访问日志
- 🚫 快速识别和封禁恶意访问
- 📊 统计访问数据
- 🛡️ 保护网站安全

## 系统要求

- 宝塔面板 7.0 或更高版本
- 已开启 API 访问
- Windows 10/11、macOS 10.15+ 或 Linux
